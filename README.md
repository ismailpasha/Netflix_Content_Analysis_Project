# Netflix Content Analysis Project

This project explores the Netflix titles dataset using Python, pandas, seaborn, and Jupyter Notebook.

## Project Files

- `notebooks/netflix_content_visualizations.ipynb`
  Main notebook with data cleaning, missing values analysis, univariate and bivariate analysis, time-series analysis, country and genre analysis, text analysis, and final observations.
- `visualize_content.py`
  Python script version for generating the visualizations.
- `netflix_titles.csv/netflix_titles.csv`
  Dataset used for the analysis.

## Main Topics Covered

- Data cleaning
- Missing values analysis
- Content type analysis
- Genre and country analysis
- Time-based trends
- Description text analysis

## Tools Used

- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## How to Run

Open the notebook:

```powershell
jupyter notebook
```

Then open:

`notebooks/netflix_content_visualizations.ipynb`

To run the Python script directly:

```powershell
python visualize_content.py
```
